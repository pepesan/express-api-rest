# Api Rest de ejemplo de MongoDB, Swagger y Mocha (Chai)

# Instalación
```shell
$ npm install
```
# Lanzamiento
```shell
$ npm start
```
[URL Principal](http://localhost:3000/)
# APIDocs
[Documentación del API](http://localhost:3000/api-docs/)

#Desarrollo
```shell
$ npm run dev
```

#Pruebas
```shell
$ npm test
```
* Nota: El directorio de los informes es mochawesome-report
